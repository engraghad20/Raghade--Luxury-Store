# Raghadé Luxury Store  

An interactive and modern  built as part of the Web Development Bootcamp journey.  
The project simulates a full online shopping experience — from browsing products to completing an order and printing a receipt.  

---

## What We Built  

-  Home / Products Page:  
  Designed an elegant interface displaying multiple products with names, prices, and ratings ⭐

- Add to Cart System: 
  Users can add products to the cart dynamically, view selected items, and update quantities.  

- Cart Summary: 
  Shows product details, total price, discounts, and tax calculations.  

- Checkout Page:  
  A side page that displays the final order summary with all prices clearly listed — subtotal, discount, tax, and total.  

- Receipt Generator: 
  Prints a detailed receipt that includes all product details, prices, totals, and discounts.  

- Responsive Design:  
  Works smoothly on all devices with a modern, clean UI.  

- Styling & Icons:  
  Custom CSS styling and **Font Awesome** icons for a polished visual touch.
